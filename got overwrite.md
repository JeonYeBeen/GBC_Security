# Got Overwrite

## Got Overwrite란? 

Dynamic Link 방식으로 컴파일된 실행 파일이 공유 라이브러리를 호출할 때 사용되는 PLT와 GOT를 이용하는 공격기법이다.

각 공유 라이브러리에 있는 함수들은 PLT 주소가 있는데, 이 주소는 GOT를 가리킨다. GOT는 함수의 실제 주소가 들어있다.

GOT의 값을 원하는 함수의 실제 주소로 바꾼다면, 바꾼 함수가 호출될 것이다.

## 실습

GDB 안에서 이루어진다.

> 참고 : https://d4m0n.tistory.com/83

### 1. GOT를 확인한다.

- `disas` 명령으로 함수를 확인하여 `PLT` 주소를 확인한다.
- `PLT` 주소를 `x/3x <주소>` 명령으로 확인하면 `jmp` 명령이 실행되는 부분에 `GOT` 주소가 있다.

### 2. 바꿀 함수의 주소를 확인한다.

- `system` 함수를 확인할 경우, `p system` 을 입력해 주소를 확인한다.

### 3. GOT에 바꿀 함수의 주소를 입력한다.

- `set` 명령어로 `GOT` 주소에 원하는 함수의 주소를 입력한다.
